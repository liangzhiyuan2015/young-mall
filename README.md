# young-mall

#### 介绍
此项目采用`SpringBoot`、`Spring Security`、`Mybatis`、`Redis`等建立起来的一整套商城，后端管理系统采用Vue，用户端采用微信小程序。用于把最近学习的东西融合到一块，未来将持续更新，后续补全文档。
